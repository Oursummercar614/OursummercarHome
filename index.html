Short info about the project

So project started on March 2025. The plan for release is to have cooperative style FP2 multiplayer with full world sync, that means each element of the game will be synchronized and both players will see the same world. I'm designing the net code to be eventually expand to support more players in future but as for now the focus is 2 players online.

If you want to follow with the team members and other project fans join official project discord. You can also hit follow button on my twitch channel where I stream development of the mod, subscribe to my youtube channel where I post videos from the development and of course hit like on projects facebook fanpage.
